
EgoRent — готовый минимальный проект для деплоя на Vercel (Next.js).

Файлы:
- package.json — зависимости и команды.
- pages/ — React-страницы (index.js содержит основной сайт).
- public/images/ — примеры изображений (svg placeholders).
- components/ui/* — простые компоненты Card и Button.
- styles/globals.css — минимальный Tailwind setup.

--- Простая инструкция (для тех, кто не программист) ---

1) Скачай архив и распакуй на компьютере.

2) Создать аккаунт на GitHub (если нет):
   - https://github.com/ — регистрация, затем нажать New repository.
   - Название репозитория: egorent

3) Залить проект на GitHub (самый простой способ — через GitHub Desktop):
   - Скачай и установи GitHub Desktop: https://desktop.github.com/
   - В GitHub Desktop: File -> Add local repository -> выбери папку распакованного проекта.
   - Commit to main -> Publish repository -> выбрать GitHub аккаунт и нажать Publish.

4) Подключить репозиторий к Vercel:
   - Перейди в https://vercel.com/ (вошёл уже).
   - New Project -> Import Git Repository -> выбери свой репозиторий egorent -> Deploy.
   - Vercel сам соберёт сайт и выдаст ссылку вида https://egorent-username.vercel.app

5) Как добавлять картинки и товары:
   - Открой папку проекта, перейди в public/images и положи туда свои картинки.
   - Зайди в GitHub Desktop -> Changes -> Commit to main -> Push.
   - Vercel автоматически задеплоит новую версию — изменения появятся на сайте.

6) Админка сейчас простая: кнопка «Админ» открывает форму добавления товара, но изменения сохраняются только в памяти (при перезагрузке или новом деплое — пропадут).
   - Чтобы данные сохранялись — позже можно подключить Supabase/Firebase. Я могу сделать это за тебя.
